### 快速开始

```
npm install docsify-cli -g

docsify init ./docs

docsify serve ./docs
```

### 分支说明

> `caddy` 分支：使用 caddy 部署，以停止维护

> `master` 分支：维护分支，使用 docsify 生成静态文件并使用 Github Pages 服务部署
